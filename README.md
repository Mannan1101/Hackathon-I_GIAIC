# Physical AI & Humanoid Robotics Textbook

A comprehensive university-level textbook on Physical AI and Humanoid Robotics, covering everything from foundational concepts to building autonomous humanoid systems.

## About This Textbook

This textbook provides a complete course on:
- **Chapter 1**: Introduction to Physical AI & Embodied Intelligence
- **Chapter 2**: ROS 2 & The Robotic Nervous System
- **Chapter 3**: Simulation with Gazebo, Unity & Digital Twins
- **Chapter 4**: NVIDIA Isaac, Perception, SLAM & Navigation
- **Chapter 5**: Vision-Language-Action & Autonomous Humanoid Capstone

## Features

- 48+ code examples (Python, C++, YAML, Bash)
- 20+ hands-on labs
- 11+ simulation activities
- 5 optional robotics experiments
- Complete capstone project

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

Using GitHub Pages:

```bash
npm run deploy
```

Or manually deploy the `build` directory to your hosting provider of choice.
